# FUTURE_ML_02 - Support Ticket Classification System

## Project Objective

This project uses Natural Language Processing (NLP) and Machine Learning to classify customer support tickets based on ticket descriptions. The system can also identify ticket priority and visualize customer support analytics.
## Technologies Used
* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Features
* Data preprocessing and cleaning
* NLP-based text vectorization using TF-IDF
* Ticket classification using Machine Learning
* Priority detection system
* Customer support analytics visualization
* Automated prediction of ticket categories
  
## Machine Learning Workflow
1. Load customer support ticket dataset
2. Clean and preprocess text data
3. Convert text into numerical features using TF-IDF
4. Train classification model using Naive Bayes
5. Predict ticket categories
6. Visualize ticket analytics using graphs

## Model Used
* TF-IDF Vectorizer
* Multinomial Naive Bayes Classifier
  
## Visualizations Included
* Ticket Category Distribution
* Ticket Priority Distribution
* Ticket Status Distribution
  
## Business Insights
1. Machine Learning can automate support ticket classification.
2. NLP reduces manual workload for customer support teams.
3. Priority detection helps resolve urgent issues faster.
4. Analytics provide insights into customer service performance.
5. Automated systems improve operational efficiency and customer satisfaction.

## Output
The system successfully classifies customer support tickets and generates meaningful analytics visualizations.
